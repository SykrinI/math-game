# math-game

## О программе
Сие суть математическая игра, написанная на Python 3. По функционалу будет похожа на Triviador с математическими задачками. 1 игрок против бота. 
## Лор
В близкой-близкой галактике живём Мы. Вдруг из мега-гиперпространства появляется **Вероятник Пифагорус Децималдор 3.1415-ый**, также известный как Вероятник Мерзейший — великий волшебник и математик с погрешностью в 0,4. Он зло-зло. Он хочет уничтожить людей своей злобной математикой, а потом, естественно, съесть мозги самых умных людей (обычай у него такой). Но нашёлся герой - Галли Поттел, который отлично знает, как бороться с математикой. Он сто тысяч миллионов раз пересмотрел всех джедаев и точно знает кто ходит в гости по утрам. Поэтому человечество надеется на этого невероятного сгустка чистой Силы. Эта битва будет легендарной!
## Участники
Проект выполняется совместно с **Сюкриным Егором**
## План
1. Сделать захват территорий.
